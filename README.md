# Java Stock Visualizer 📈

A real-time desktop application that tracks and visualizes the Dow Jones Industrial Average (^DJI). Built with **Java**, **Gradle**, and **JavaFX**.

## 🚀 Features
- **Live Data:** Queries Yahoo Finance API every 5 seconds.
- **Visuals:** Renders a dynamic line chart using JavaFX.
- **Resilient:** Handles network errors and API rate limits automatically.

## 🛠️ Built With
- **Java 21**
- **Gradle** (Build Tool)
- **YahooFinanceAPI** (Data Source)
- **JavaFX** (GUI)

## 💻 How to Run
1. Clone the repo.
2. Run `gradle run`.
